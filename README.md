# SKALA: Grain Segmentation & Classification Library

Библиотека для сегментации и классификации гранул на микроизображениях с использованием глубокого обучения.

# Перечень направлений прикладного использования

1. **Автоматизация петрографического анализа в геологоразведке**  
   Быстрое определение минерального состава песчаников (12 классов по диаграмме Шутова) по шлифам, что ускоряет оценку перспективности месторождений полезных ископаемых (нефть, газ, подземные воды, строительные материалы).
2. **Количественная оценка коллекторских свойств горных пород**  
   Использование инстанс-сегментации (Mask R-CNN) для выделения отдельных зёрен кварца и других минералов с последующим расчётом пористости, степени цементации и контактов зерен — критично для моделирования нефтегазовых резервуаров.
3. **Стандартизация и ускорение экспертизы в кернохранилищах**  
   Замена трудоемкого ручного подсчёта зёрен (сотни шлифов в день) на автоматический пайплайн, снижение субъективизма и разногласий между экспертами (ОТАР/ОПАР >1 свидетельствует о превосходстве над межэкспертным согласием).
4. **Обучение студентов-геологов и повышение квалификации**  
   Интерактивная демонстрация эталонной сегментации и классификации шлифов, сравнение с результатами экспертов — наглядное пособие по петрографии и оптической минералогии.
5. **Создание цифровых двойников керна**  
   Генерация 3D-структуры по сегментированным 2D-срезам (алгоритмы Любачевского–Стиллинджера, гравитационная насыпка) для физически достоверного моделирования фильтрации флюидов и распространения упругих волн.
6. **Оптимизация технологии обогащения руд**  
   Автоматическое распознавание срастаний и сростков минералов в шлифах для расчета степени раскрытия зёрен — входные данные для настройки режимов дробления и измельчения.
7. **Интеграция в лабораторное оборудование**  
   Встраивание библиотеки в ПО оптических микроскопов (например, Биоптик ВР-200) для выдачи результатов в реальном времени: «образец — процент кварца, полевых шпатов, обломков пород и т.д.».
8. **Бенчмарк для научных исследований**  
   Открытая библиотека с публичными интерфейсами и синтетическим датасетом мультимодальных изображений (параллельные/скрещенные николи) — инструмент для сравнения новых архитектур сегментации (U-Net, DeepLab и др.) на реалистичной физически-информированной модели поляризованного света.
9. **Создание отраслевых нормативных баз данных**  
    Формирование стандартизированных коллекций сегментированных шлифов в форматах JSON/CSV для обучения будущих моделей и сертификации программных средств в геологической отрасли (аналог «базы эталонных шлифов ГКЗ»).


## Минимальные технические требования для запуска и использования открытой библиотеки

- Операционная система Linux или Windows Subsystem for Linux с Ubuntu >=20.04
- CPU 13th Gen Intel(R) Core(TM) i5-13600KF 3500 Mhz или лучше
- RAM >= 16 Gb
- GPU >= 12 Gb памяти (например, NVIDIA GeForce RTX 4070 или лучше)
- CUDA >= 13.2
- Требования по версиям python и пакетов приведены в pyproject.toml

## Использование

```python

# Расчет метрик сегментации

    folder=Path("data/slices"),

# {'precision': 0.85, 'recall': 0.82, 'iou': 0.73, 'f1': 0.83}
```

## Модели

### Mask R-CNN
- Детекция и сегментация гранул
- Модифицированные anchor boxes
- Замороженный backbone

### U-Net
- Семантическая сегментация
- Skip-connections
- Выход: бинарная маска

### Классификаторы
- ResNet50 (полная заморозка)
- EfficientNet-B0 (частичная заморозка)

## Метрики

### Сегментация
- **IoU** (Intersection over Union)
- **F1-score**
- **Precision / Recall**
- **Accuracy**

### Классификация
- **OTAR** (Operator True Accept Rate)
- **OPAR** (Operator Precision Accept Rate)
- **Precision / Recall** (macro average)
